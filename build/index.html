About Project

  • Focus Commercial Cleaning UI/UX
  • Scope UI/UX + Branding
  • Date Aug - Nov 2018

As a busy executive, finding a commercial cleaner is time consuming. The current user expereince is text heavy, and hard to navigate. This projects aims to understand how executives are making decisions and improving the user expereince

User Research

During this user research I interviewed four executives to understand how they are making decisions when they are choosing a commercial cleaner.

User journey

To understand the user journey I conducted user testing on three different cleaning websites to discover how they navigate the content.

Content Strategy

The user research led to many informed decisions about the content strategy and site map. This included making sure there is a clear heirachy, POD, and clear categorization. This helped inform the sitemap.

Branding

The colour purple is used to champion a Premium feel. The monochromatic imagery plays on this aswell and allows for a high contrast colour pallette for easier discovery.
